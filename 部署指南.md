# 部署到 GitHub Actions（7×24 免费运行）

本工具已适配 GitHub Actions：每 30 分钟自动算信号，有买入/卖出信号就发邮件给你。
全程免费、无需服务器、无需绑卡，只要一个 GitHub 账号。

---

## 前提
- 一个 GitHub 账号（没有就去 github.com 注册，免费）

## 步骤

### ① 在 GitHub 新建公开仓库
1. 登录 github.com → 右上角 **+** → **New repository**
2. 名字随便，比如 `stock-signal`
3. 选 **Public**（公开仓库 Actions 分钟数无限；私有也行但免费额度有限）
4. **不要**勾选 "Add a README" 等任何选项
5. 点 **Create repository**
6. 复制仓库地址（形如 `https://github.com/你的用户名/stock-signal.git`）

### ② 把代码推上去
本地仓库已经帮你 `git init` + `commit` 好了，只需在 **终端** 执行（把 URL 换成你的）：

```bash
cd /Users/zhangzhenyu/WorkBuddy/2026-07-16-14-06-11/招商证券策略提醒工具
git remote add origin https://github.com/你的用户名/stock-signal.git
git branch -M main
git push -u origin main
```

**push 时要输入的"密码"不是 GitHub 登录密码，而是 Personal Access Token：**
- GitHub → 右上角头像 → Settings → Developer settings → Personal access tokens → Tokens (classic)
- Generate new token → 勾选 **repo** 权限 → 生成 → 复制那串 token
- push 时用户名填你的 GitHub 用户名，密码处粘贴这个 token
- （push 成功后这个 token 可以删除，不影响已部署的 Actions）

### ③ 配置邮箱 Secrets（让代码能发邮件，且不泄露授权码）
1. 仓库页面 → **Settings** → **Secrets and variables** → **Actions**
2. 点 **New repository secret**，添加两个：
   - Name：`MAIL_EMAIL`　　Value：`zhangzhenyu.chn@qq.com`
   - Name：`MAIL_AUTHCODE`　Value：`ftkrlydevxotbdgf`
3. 添加后 Secrets 显示为隐藏，安全

### ④ 手动触发一次，验证跑通
1. 仓库 → **Actions** 标签 → 左侧选「策略信号定时扫描」
2. 右侧点 **Run workflow** → 运行
3. 等 1-2 分钟，点进这次运行记录看日志
   - 看到「无新信号，不发邮件」或「邮件发送成功」→ 跑通了 ✅
   - 如果看到拉数据报错 → 可能是美国服务器访问国内行情偏慢，多触发几次看是否恢复

### ⑤ 之后全自动
- 每 30 分钟自动运行一次
- 有买入/卖出信号才发邮件，同信号当天只发一次（不刷屏）
- 你的 Mac 关机也没关系，跑在 GitHub 服务器上

## 常见问题

**Q：想换自选股怎么办？**
A：仓库里编辑 `watchlist.txt`，每行一个代码，提交即可。下次运行生效。

**Q：定时不准时？**
A：GitHub 的 cron 在高峰期可能延迟几分钟到十几分钟，对 A 股信号没影响。

**Q：美国服务器拉国内行情会不会不通？**
A：腾讯/东方财富行情接口是全球可访问的，只是从美国访问偏慢。代码已有 8 秒超时 + 重试，绝大多数情况能拉到。若长期拉不到，再考虑换国内云方案。

**Q：周末/节假日也跑吗？**
A：会跑，但行情不变、信号不变，防重复逻辑会自动跳过发信，不会打扰你。

**Q：想停掉？**
A：仓库 Settings → Actions → 把 workflow 禁用即可。
